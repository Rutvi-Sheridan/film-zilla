# Overview

Film Zilla is a front-end web design project developed using **HTML5** and **CSS3**. The website showcases a modern streaming platform interface with an emphasis on clean design, intuitive navigation, and responsive layouts.

The project demonstrates front-end development fundamentals and UI design principles while providing users with an engaging browsing experience.
